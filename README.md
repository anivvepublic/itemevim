# Itemevim - Dijital Ürün Pazar Yeri

Oyun hesapları ve sosyal medya hesapları için güvenli alım-satım platformu.

## 🚀 Teknolojiler

- **Frontend:** React 18 + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **Auth:** Supabase Auth
- **Real-time:** Supabase Realtime

## 📦 Kurulum

### Frontend
```bash
cd frontend
npm install
npm run dev
